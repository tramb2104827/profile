# PDF Documents Folder

Place your PDF documents in this folder:

- `CV.pdf` - Curriculum Vitae
- `Transcript.pdf` - Academic Transcript
- `Achievements.pdf` - Achievements Document
- `Recommendation.pdf` - Recommendation Letter (Optional)

## How to add PDF files:

1. Place your PDF files in this `public/pdf/` folder
2. Make sure the filenames match exactly:
   - CV.pdf
   - Transcript.pdf
   - Achievements.pdf
   - Recommendation.pdf

3. The download functionality will automatically work once files are added.

## Note:

If files are not found, users will see a helpful error message. Make sure to:
- Use lowercase filenames
- Keep the exact filenames as listed above
- Ensure files are PDF format
