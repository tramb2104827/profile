# Personal Academic Portfolio Website

A modern, responsive academic portfolio website built with React for scholarship application (bilateral scholarship to Russia).

## Features

- **Modern Design**: Clean, academic-focused design with white and turquoise color scheme
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Smooth Navigation**: Fixed navigation bar with smooth scrolling
- **Semantic HTML**: Properly structured for SEO
- **Fast Loading**: Optimized performance with minimal animations

## Sections

1. **Home**: Hero section with introduction and call-to-action buttons
2. **About Me**: Personal background and information
3. **Education**: Academic qualifications and focus areas
4. **Experience**: Teaching and professional experience
5. **Projects**: Key projects and technologies used
6. **Research Interests**: Academic research focus areas
7. **Achievements & Activities**: Extracurricular and leadership roles
8. **Portfolio**: Download links for CV and academic documents
9. **Contact**: Contact information and email

## Technologies

- React 18.2
- HTML5 & CSS3
- Font Awesome Icons
- Google Fonts (Inter)

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.

## Deployment

The site can be deployed to:
- **GitHub Pages**: Use `gh-pages` package
- **Netlify**: Connect to GitHub repository for automatic deployments
- **Vercel**: Import project and deploy
